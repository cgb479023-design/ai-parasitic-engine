## 📊 三大算法门槛 (严格筛选标准)

### 🚪 GATE 1: 选择观看率 (Viewed vs. Swiped Away)
这是算法决定是否把视频推送到更大流量池的"入场券"。

| 级别 | 观看率 | 预期播放量 | 状态 |
|------|--------|------------|------|
| 及格线 | 60-70% | 1万-5万 | ⚠️ 边缘 |
| **爆发线** | **70-80%** | **10万-100万+** | ✅ **目标** |
| 神级线 | >80% | 1000万+ | 🔥 极限 |

**当前基线**: 58.4% (低于及格线边缘)
**目标**: 提升至 **75%+**

---

### 🔄 GATE 2: 平均观看百分比 (APV - Average Percentage Viewed)
这是决定算法推流"持久度"的核心。**9秒视频要求极其严苛。**

| 表现 | APV | 说明 |
|------|-----|------|
| 失败 | 80-90% | 大多数人没看到结尾就划走 |
| 普通 | 100% | 刚好看完一遍 |
| **病毒式** | **>120%** | 观众复看 (Loop) ✅ **必须达到** |

**原理**: 对于超短视频，算法要求观众不仅看完，还要**"复看" (Loop)**。
**解决方案**: 视频结尾必须与开头无缝衔接，或者信息量大到观众需要看第二遍才能看懂。

---

### ❤️ GATE 3: 互动率 (Engagement Rate)
辅助助推器，加速进入更大流量池。

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 点赞率 (Like/View) | **5-10%** | 每100观看至少5-10个赞 |
| 评论率 | **>1%** | 争议性问题驱动评论 |
| 分享率 | >0.5% | "Tag a friend" 策略 |

## 🎯 "3-2-1" 病毒公式 (最快通关策略)

### 🔴 3-SECOND PRINCIPLE (攻克观看率 → 75%+)
第0-1秒必须出现强烈视觉刺激，阻止划走。

**执行要点**:
- Frame 0-0.3s: **RED CIRCLE** 或 **ARROW** 指向异常点
- Frame 0.3-1s: 建立 "这里有问题?" 的悬念
- Frame 1-3s: 紧张升级，观众无法划走
- **禁止**: 慢速开场、Logo、文字铺垫

**脚本模板**:
```
⚠️ 'You won't believe what we found...' (悬疑音效 + 文字叠加)
🔍 'The TRUTH they tried to hide...' (镜头推进 + 红圈高亮)
😱 'This changes EVERYTHING...' (快剪 + 低音炮)
```

---

### 🔁 2-LOOP PRINCIPLE (攻克APV → 120%+)
设计无缝循环，让观众不知不觉看第二遍。

**执行要点**:
- **禁止**: 黑屏结尾、淡出效果
- **必须**: 最后一帧动作与第一帧动作无缝衔接
- **示例**: 视频结尾人物跑出画面 → 画面切回开头人物刚进入画面（同样的运动模糊）

**循环触发器设计**:
```
Frame 7.5s: 人物向右跑出画面
Frame 0.0s: 人物从左侧进入画面（同样的动作姿态）
观众效果: 无意识中观看1.2-1.5遍
```

---

### ❓ 1-CONTROVERSY PRINCIPLE (攻克评论区)
故意埋藏一个隐藏细节，引导回看和评论。

**隐藏细节策略**:
- 故意拼错的文字 (如 "EVIDANCE" 而非 "EVIDENCE")
- 背景里有个奇怪的人影
- 时间戳显示不可能的时间
- 文件名包含隐藏信息

**置顶评论模板** (Mark Bobl 人设):
```
⚠️ Case #[随机编号] Analysis:

👀 Did anyone notice the spelling at 0:04? Something doesn't add up...

📊 Drop your theory below - what REALLY happened here?

(Follow for daily forensic investigations 🔍)
```

**目标**: 让观众回去重看（增加 APV）并留言讨论。
