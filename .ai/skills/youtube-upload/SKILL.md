# YouTube上传技能包

## 1. 概述

本技能包实现了YouTube视频上传和发布功能，支持自动上传、填写元素、设置发布时间和状态同步。

## 2. 功能模块

### 2.1 视频上传
- **描述**：将生成的视频上传到YouTube
- **特点**：支持大文件上传，显示上传进度
- **使用场景**：单个视频上传，批量视频上传

### 2.2 元素填写
- **描述**：自动填写视频标题、描述、标签等元素
- **特点**：支持模板化填写，智能生成标签
- **使用场景**：标准化视频发布，批量视频发布

### 2.3 发布设置
- **描述**：设置视频的发布时间和隐私状态
- **特点**：支持定时发布，支持隐私设置
- **使用场景**：预约发布，隐私视频发布

### 2.4 状态同步
- **描述**：将视频状态同步到React发布计划
- **特点**：实时同步，支持状态回传
- **使用场景**：视频发布状态跟踪，工作流闭环

## 3. 工作流阶段

### 阶段一：Spec生成与对齐

1. **输入**：YouTube上传需求（如"上传视频并设置定时发布"）
2. **操作**：
   - 确定上传参数
   - 生成上传配置
   - 定义验收标准
3. **输出**：`specs/YOUTUBE_UPLOAD_spec.md`
4. **网关**：需人类开发者输入"Approved"方可进入下一阶段

### 阶段二：脚本化执行

1. **上传视频**：调用视频上传脚本上传视频
2. **填写元素**：自动填写视频标题、描述、标签等
3. **设置发布时间**：根据React发布计划中的时间设置YouTube自动发布
4. **等待上传完成**：等待视频100%上传完成

### 阶段三：自动化验证

1. **验证上传成功**：检查视频是否上传成功
2. **验证发布设置**：检查发布时间和隐私状态是否正确
3. **验证状态同步**：检查视频状态是否同步到React发布计划

## 4. 脚本工具

### 4.1 视频上传脚本

**文件**：`scripts/upload-video.py`

**功能**：上传视频到YouTube

**参数**：
- `--video-file`: 视频文件路径
- `--title`: 视频标题
- `--description`: 视频描述
- `--tags`: 视频标签（逗号分隔）
- `--publish-time`: 发布时间（ISO格式）
- `--privacy-status`: 隐私状态（public/unlisted/private）

### 4.2 状态同步脚本

**文件**：`scripts/sync-status.py`

**功能**：将视频状态同步到React发布计划

**参数**：
- `--video-id`: 视频ID
- `--status`: 视频状态（scheduled/published）
- `--publish-time`: 发布时间（ISO格式）

## 5. 验收标准

### 5.1 视频上传
- 视频成功上传到YouTube
- 上传进度显示正确
- 支持断点续传

### 5.2 元素填写
- 视频标题、描述、标签填写正确
- 支持自动生成标签
- 支持模板化填写

### 5.3 发布设置
- 发布时间设置正确
- 隐私状态设置正确
- 支持定时发布

### 5.4 状态同步
- 视频状态成功同步到React发布计划
- 支持状态回传
- 实时同步

## 6. 最佳实践

### 6.1 视频优化
- 视频分辨率：建议1080p或更高
- 视频格式：MP4格式
- 视频编码：H.264编码
- 音频编码：AAC编码

### 6.2 元数据优化
- 标题：清晰明确，包含关键词
- 描述：详细描述，包含关键词和链接
- 标签：相关关键词，5-10个
- 缩略图：高质量，吸引人

### 6.3 发布策略
- 发布时间：根据目标受众的活跃时间设置
- 隐私状态：根据需求设置
- 定时发布：提前规划发布时间

## 7. 故障排除

### 7.1 上传失败

**症状**：视频上传失败

**解决方案**：
1. 检查网络连接
2. 检查视频文件格式
3. 检查YouTube API配额
4. 检查视频大小限制

### 7.2 发布时间设置失败

**症状**：发布时间设置失败

**解决方案**：
1. 检查时间格式
2. 检查时区设置
3. 检查API权限

### 7.3 状态同步失败

**症状**：视频状态同步失败

**解决方案**：
1. 检查网络连接
2. 检查React发布计划服务
3. 检查状态同步脚本

## 10. 使用示例

### 10.1 上传视频

```bash
python scripts/upload-video.py \
  --video-file output/creative-video.mp4 \
  --title "AI创意视频" \
  --description "这是一个关于AI的创意视频" \
  --tags "AI,创意,视频" \
  --publish-time "2026-01-26T10:00:00+08:00" \
  --privacy-status "public"
```

### 10.2 同步视频状态

```bash
python scripts/sync-status.py \
  --video-id "test-video-1" \
  --status "scheduled" \
  --publish-time "2026-01-26T10:00:00+08:00"```
